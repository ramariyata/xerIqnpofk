# 项目简介

## 核心功能
- 支持吸烟行为和人类检测识别
- 提供图片、视频、摄像头三种检测方式
- 支持自定义训练模型，可识别多种动物种类
- 实时预测信息展示，包括预测结果和置信度
- 详细记录预测时间、预测图片和操作记录
- 分用户管理系统，管理员可查看所有记录
- 报警功能，检测到危险行为时触发报警
- 邮件通知功能，与报警系统联动发送邮件

## 适用场景
- 公共场所吸烟行为监控
- 安全监管系统
- 行为分析研究
- 动物种类识别研究
- 实时视频监控分析

## 技术特点
- 前端采用Vue3框架开发
- 后端使用Django框架
- 支持多种YOLO模型版本（yolov8、yolov10、yolo11、yolov12）
- 数据库支持MySQL和sqlite
- 界面支持主题颜色和布局自定义
